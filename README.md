# Агентство X

Лендинг и брендбук проекта "Агентство X" про AI-аватаров, deepfake-production и генерацию реальности для маркетинга и контента.

## Структура

- `index.html` - главная страница лендинга
- `styles.css` - единственный актуальный файл стилей главной страницы (альтернатива `homepage-styles.css` в проекте не используется и отсутствует)
- `brandbook.html` - страница брендбука
- `brandbook-styles.css` - стили брендбука

## Запуск локально

Откройте файл `index.html` в браузере двойным кликом или через Live Server в редакторе.

## Публикация на GitHub Pages

1. Создайте репозиторий и загрузите в него файлы проекта в корень.
2. В GitHub откройте `Settings -> Pages`.
3. В разделе **Build and deployment** выберите:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (или `master`)
   - **Folder:** `/ (root)`
4. Нажмите **Save** и дождитесь публикации.
5. GitHub выдаст ссылку вида `https://<username>.github.io/<repo>/`.

## Важно для корректной работы на GitHub Pages

- В проекте используются только относительные пути к локальным файлам.
- Абсолютные локальные пути Windows (например `C:/Users/...`) не используются.
- Все файлы в именах используют lowercase и дефисы, что совместимо с хостингом GitHub Pages.
